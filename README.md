### 구현 사항
<hr />
<li>Toggle : 토글 스위치 ON / OFF 기능 구현</li>
<li>Modal : Open 버튼 클릭 시 모달창 노출 기능 구현</li>
<li>Tab : 각 Tab 버튼 클릭 시 정보 노출 기능 구현</li>
<li>Tag : Enter 키 클릭 시 Tag 등록되도록 기능 구현</li>
<li>ClickToEdit : 이름과 나이 입력 시 정보 노출 기능 구현</li>
